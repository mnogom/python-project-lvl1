## Instalation
```bash
$ make build # builds the source and wheels archives
$ make package-install # pip install
```


## Call games
> Be sure that /Users/<username>/Library/Python/3.7/bin in $PATH
```bash
$ brain-even
$ brain-calc
$ brain-gcd
$ brain-progression
$ brain-prime
```
